Danmuplayer

View project on GitHub

Feature

  • Web video player with scrolling comments
  • Store users' comment in database
  • Load all the stored comment each time the video is played
  • Flexible style of comments including color, size, position, transparency, etc.

Detail

Danmuplayer is a mp4 player which is written in JS and used by Jquery. In china streaming platform and video website, this is a very popular kind of player for users. It provides vary choice of comment style and user can focus on the video without looking for the comment aside.


This project is under studying.The showing picture is my danmuplayer workspace running in cloud 9. The danmuplayer itself is based on the open source code. What I have done includes creating the database and completing the query in php to get and load comments from database.
